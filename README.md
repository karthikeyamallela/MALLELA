# Smart Weather Monitoring System

This project turns your weather dashboard design into a complete full-stack application with persistent local storage.

## Stack

- Frontend: HTML, CSS, vanilla JavaScript
- Backend: Node.js + Express
- Database: file-backed JSON datastore

## Features

- Login with demo credentials
- Dashboard metrics loaded from backend APIs
- Live weather updates every 5 seconds
- Sensor CRUD with persistent storage
- Weather history, alerts, devices, and settings pages
- Persistent datastore created automatically in `data/weather-db.json`

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`
